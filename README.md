本地账号密码管理器：安全管理账号密码，就这么简单！给大家推荐一款超实用的本地账号密码管理器，主打本地存储、安全加密，不用联网同步，数据牢牢握在自己手里。
<img width="1282" height="846" alt="QQ20251114-092944" src="https://github.com/user-attachments/assets/7a23cc4c-cdaa-48fc-958f-2e38442b196f" />

它采用 AES - 256 加密技术，把保存的数据统一加密，解密密钥还单独存放在文件里，双重保障，安全感直接拉满。首次启动时，会自动生成secret.key和saved_accounts.json文件，就在软件同目录下，方便查找和备份。要是担心启动安全，还能在 “软件加密” 里设置启动密码，设置后每次启动都得输入密码，进一步筑牢安全防线。
<img width="376" height="299" alt="QQ20251114-093026" src="https://github.com/user-attachments/assets/2b24d690-e766-4b7b-afb6-9410d15bb487" />

功能方面特别丰富。自动生成账号密码超方便，能根据长度和字符规则生成，大小写、数字、符号这些都能自由搭配。手动录入也很简单，网站、账号、密码、手机、邮箱、备注这些信息，一键就能保存。搜索功能也很给力，输入网站、账号、邮箱、手机号或者备注里的关键词，就能快速过滤出想要的记录。而且支持 CSV 格式的导入导出，备份和迁移数据轻松搞定。另外，点击 “使用必读” 能查看安全声明和注意事项，想支持开发者的话，点 “赞助支持” 就能看到赞助图片，窗口底部的 “荣誉出品” 还能直达 B 站主页。
<img width="502" height="659" alt="QQ20251114-093046" src="https://github.com/user-attachments/assets/c4cf41ae-bd79-48bf-9d5a-934b67df6161" />

使用起来也不难。添加记录有两种方式，要么在自动生成区设置好规则，点击 “开始生成”，再点 “保存录入”；要么在手动录入区填完信息直接点 “保存录入”。搜索的时候，在搜索框输入关键词点 “搜索” 就行，勾选 “显示密码” 还能切换密码的显示或隐藏状态。数据管理上，选中行点击 “删除选中记录” 会永久删除，没法恢复，所以删除得谨慎。“导出 CSV” 能生成备份文件，“导入数据” 可以按 CSV 字段导入，不过得至少包含account和password。安全设置里，“软件加密” 能设置或取消启动密码，取消时留空确认就可以。
